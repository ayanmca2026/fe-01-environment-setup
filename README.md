## 📌 Project Overview
The Frontend AI Engineering Toolkit is a foundational repository demonstrating best practices for integrating AI coding assistants (like Cursor or Claude Code) into modern frontend development workflows. This project serves as the baseline environment setup for the FE-01 assignment.

## ✨ Features
- **AI-Optimized Context**: Includes `CLAUDE.md` to feed critical project context to AI assistants.
- **Clean Git Workflow**: Pre-configured `.gitignore` tailored for Node.js, React, and VS Code.
- **Open Source Ready**: Licensed under the MIT License for permissive usage.
- **Standardized Practices**: Enforces coding conventions and folder structures suitable for AI-collaborative development.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS version recommended, e.g., v20.x or v22.x)
- [Git](https://git-scm.com/)
- An AI-powered IDE or CLI tool: [Cursor](https://cursor.sh/) or [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/fe-01-environment-toolkit.git
   ```
2. Navigate to the project directory:
   ```bash
   cd fe-01-environment-toolkit
   ```
3. Open the project in your AI-assisted IDE (e.g., Cursor):
   ```bash
   cursor .
   ```

## 🤖 AI Toolchain Integration
This project is designed to be developed alongside AI tools. The `CLAUDE.md` file acts as the "system prompt" for AI agents operating within this repository, providing them with architectural guidelines, tech stack details, and coding conventions.
